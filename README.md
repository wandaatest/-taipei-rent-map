# 台北女子圖鑑租屋地圖

一個以生活風格為出發點的台北租屋 MVP 網站。

這不是房仲資料庫，也不是單純比較哪一區最便宜，而是想用更接近日常的方式，回答一個更私人的問題：

**在台北，你想把日子過成什麼樣子？**

使用者可以透過 5 題生活風格測驗，得到最適合自己的台北生活圈推薦，並在地圖上快速閱讀每個區域的租金感受、街區氛圍與生活提案。

## 網站內容

目前收錄 6 個台北生活圈：

- 大安／科技大樓
- 中山／雙連
- 信義／市政府
- 萬華／西門
- 士林／天母
- 文山／景美木柵

主要頁面：

- `Home`：首頁與 6 區預覽
- `Quiz`：5 題單選生活風格測驗
- `Result`：人格類型、首選區域與次推薦區域
- `Map`：互動式租屋地圖與區域生活卡片

## 專案特色

- Vue 3 + Vite + TypeScript
- Tailwind CSS
- Vue Router
- 寫死資料的 MVP 架構，無需後端
- 雜誌感、女性生活風格編輯視角的 UI
- GitHub Pages 自動部署

## 本機開發

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build
```

## 部署

本專案已設定 GitHub Actions，自動部署到 GitHub Pages。  
只要 push 到 `main`，就會自動重新 build 與發佈。

## 圖片與字體

- 首頁背景圖使用 Unsplash 圖片，頁面上已附攝影師署名
- 顯示字體使用 `Noto Serif TC` webfont，確保手機上也能穩定呈現中文 serif 風格

## 備註

目前站內的租金、生活圈描述、店家與生活提案多為 MVP 階段的編輯型假資料，後續可再替換成正式街區照片、真實地址與最終文案。
